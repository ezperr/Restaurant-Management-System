Курсовая работа на тему "Разработка БД и приложения для автоматизации деятельности предприятия общественного питания"
Restaurant Order Management System

О проекте:
Десктопное приложение для автоматизации управления заказами в ресторане.
Позволяет вести учёт столиков, меню и заказов.

Технологии:
- C# / Windows Forms
- MySQL
- ADO.NET (MySql.Data)

Функционал:
- Управление столиками (CRUD)
- Управление меню (CRUD + валидация цен)
- Создание и редактирование заказов
- Автоматический расчёт суммы заказа
- Автоматическое обновление статусов столиков
